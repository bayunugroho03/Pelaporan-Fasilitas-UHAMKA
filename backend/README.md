# Backend Lapor UHAMKA

## Cara Installasi
1. Pastikan XAMPP berjalan (Apache & MySQL).
2. Buat database `uhamka_lapor_db` di phpMyAdmin.
3. Masuk ke folder backend: `cd backend`.
4. Install module: `npm install`.
5. Jalankan server: `nodemon index` atau `node index`.

## Fitur
- Simulasi Email Verifikasi (Cek Console Log saat Register).
